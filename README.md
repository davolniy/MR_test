Тестовое приложение. Отображает список репозиториев (gitHub api) gitHub с возможностью детального просмотра каждого через webView.

Архитектура построена на принципах Clean Architecture

Основной паттерн MVP

Стек:
  1. Kotlin
  2. Retrofit2
  3. OkHttp3
  4. RxJava3
  5. Moxy
  6. Cicerone
  7. Toothpick
